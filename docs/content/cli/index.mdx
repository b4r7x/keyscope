---
title: CLI
description: Command reference for the keyscope CLI.
---

The keyscope CLI installs and manages keyboard hooks as local source files.

## Commands

| Command | Description |
| --- | --- |
| `init` | Initialize `keyscope.json` in a project |
| `add` | Add one or more hooks |
| `list` | List available or installed hooks |
| `diff` | Compare local hook files with registry versions |
| `remove` | Remove installed hooks |

Use `npx keyscope <command> --help` for full flag details.
